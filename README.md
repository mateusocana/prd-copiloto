# OpenClaw + Obsidian — Cérebro Externo do Everton Rosa

Este vault é o cérebro externo do ecossistema do Everton Rosa.

Não é um backup passivo. É a base viva de contexto, memória, biblioteca-fonte, método e operação do OpenClaw.

## Áreas do vault

- `memory/` memória viva, decisões, prioridades, automações candidatas, projetos, linguagem, casos e pendências
- `mocs/` mapas canônicos para navegação e pensamento
- `concepts/` conceitos-base do posicionamento, autoridade e valor percebido
- `dashboards/` painéis das frentes principais
- `profiles/` recortes operacionais do perfil do Everton
- `proof/` provas, casos e ativos de credibilidade
- `playbooks/` rotinas repetíveis com posicionamento
- `library/books/` livros-fonte do Everton Rosa
- `library/method/` ebooks e módulos do método
- `scripts/` automações operacionais
- `.obsidian/` interface, layout e preferências do vault

## Regra do sistema

Se um contexto importa, ele deve existir aqui de forma conectada.

Arquivos soltos viram ruído.
Nós conectados viram inteligência.

## Manutenção de saúde

- auditoria automática do grafo em [[logs/graph-health]]
- recompilação de [[memory/index]] a cada hora
- nenhuma captura relevante deve ficar fora da malha de memória
